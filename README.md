# دیابت‌یار (Diabetes Yar) — وب‌اپ فارسی

این نسخه یک **MVP واقعی** است برای:
- ثبت قند خون (فرمت خانواده‌ها: FBS، ۲ساعت بعد صبحانه، قبل/بعد ناهار، قبل/بعد شام، قبل خواب)
- ثبت انسولین (Basal/Bolus/Correction)
- ثبت وعده غذایی + کربوهیدرات + لینک به قند ۲ ساعت بعد
- داشبورد (نمودار ۷ روز اخیر + KPIها + هشدارهای ایمنی)
- چند پروفایل (چند بیمار)
- یادآور Basal با **Notification**
- خروجی **PDF**
- صفحه آموزش + بانک غذای نمونه

## اجرا (لوکال)
فایل `index.html` را باز کنید یا یک سرور ساده اجرا کنید:

```bash
python -m http.server 8000
# سپس:
# http://localhost:8000
```

## GitHub Pages
در Settings → Pages، شاخه `main` و مسیر `/(root)` را انتخاب کنید.

## نکته ایمنی
این ابزار **پیشنهاد درمان/دوز** ارائه نمی‌دهد و جایگزین پزشک نیست.
